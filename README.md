# SCALE documentation

## Generating public documentation

Dependencides:

```sh
yay -S mkdocs pymdown-extensions
pip install mkdocs-gitbook
```

Development:

```sh
mkdocs serve
```

Generating:

```sh
mkdocs build
```

Good example for features:
- https://example-mkdocs-basic.readthedocs.io/en/latest/
